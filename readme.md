# KeyVoid

A music community platform for discovering emerging artists and expanding music taste.

Features:
- Creator system
- Media player
- Key Void discovery mode

Backend env:
- Set `ADMIN_EMAIL` in `KEYVOID/BACKEND/.env` to your own email address.
- Accounts with that email are promoted to `admin` by the backend.
- Public signup only allows `user` and `creator`, never `admin`.
